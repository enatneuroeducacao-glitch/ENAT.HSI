# Central ENAT HSI — Minimal React App

Scaffold mínimo com Vite + React contendo o componente `CentralENATHSI` (versão sem dependências de UI externas).

Instalação e execução:

```bash
cd /workspaces/ENAT.HSI
npm install
npm run dev
```

O servidor de desenvolvimento aparecerá em `http://localhost:5173` por padrão.

Arquivos principais:
- `src/App.jsx` — componente principal e páginas.
- `src/main.jsx` — ponto de entrada.
- `index.html` — entrada do Vite.
